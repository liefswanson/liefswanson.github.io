This page contains a list of citations for use in IAT339. This page does not exist in the production site at lief.info.

Many of the images on this site came from websites like pexels, unsplash, and pixabay. Some of the images have since been removed... I am not sure if this meant they were uploaded in violation of copyright, or as some of these stock photo sites are supported by paid stock photo companies... maybe they were removed so that future people looking would be given a paid alternative suggestion.

You can also find links to prerendered pages of the site here, to give something to run through total validator. These prerendered pages are not meant to be used, just validated. They don't work as none of the javascript is packaged with them, so they are completely non-functional. There are 5 main classes of validation problems I have not been able to fix.

  1. The url for google fonts contains an invalid url character by design.
  2. Tags inserted by webpack (such as style and script tags) contain type attributes, which is no longer needed in the current html standard... but doesn't hurt anything at all.
  3. H1 elements are not technically top level, because they can't be given the way vue treats templates.
  4. Because my css is generated from scss, many vendor prefixes are injected that validators don't recognize.
  5. Vue injects a span with id="warning-container" before the body element once it starts, allowing it to display warnings, this will actually cause the W3C validator to stop reading the HTML, so I manually removed these in my prerendered versions.
